# Critical SOPs

Files dropped in this folder are indexed as a **single atomic
chunk** — partial retrieval is impossible.  Use this for:

- Incident-response runbooks
- Settlement / trade rollback procedures
- KYC red-flag lists
- Regulatory disclosure checklists
- Any procedure where a missing step is a compliance incident

See `docs/critical-blocks.md` for the full spec, including the
inline `[[CRITICAL: title]]...[[/CRITICAL]]` marker that works
inside ordinary documents.
